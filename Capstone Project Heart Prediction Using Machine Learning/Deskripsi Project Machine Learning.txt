Proyek ini bertujuan untuk mengembangkan model machine learning untuk 
mendeteksi risiko penyakit jantung berdasarkan atribut medis seperti usia, tekanan darah, kolesterol, dan lainnya. 
Prosesnya mencakup:
- Memahami atribut dataset
- Preprocessing
- Melakukan modeling menggunakan minimal tiga algoritma klasifikasi, membandingkan performanya, dan melakukan tuning parameter untuk meningkatkan kinerja
- Mengevaluasi model dengan metrik seperti accuracy, precision, recall, dan F1-Score, serta memilih model terbaik berdasarkan F1-Score.
Terakhir saya, membuat API sederhana menggunakan Flask untuk memanfaatkan model terbaik yang didapatkan dalam klasifikasi berbasis input.
Sehingga output yang dihasilkan meliputi:
- Laporan analisis lengkap dalam format PDF.
- Source code analisis (data understanding hingga evaluasi).
- Source code untuk API model deployment.
